# OrbitBook 🚀🌌

Uma plataforma revolucionária para reserva e gerenciamento de experiências de turismo espacial.

## 📖 Sobre o Projeto

**OrbitBook** é uma plataforma digital desenvolvida para atender o futuro mercado de turismo espacial, oferecendo um ambiente centralizado para gerenciamento e reserva de experiências fora da Terra.

O projeto se destina a conectar usuários interessados em viagens espaciais com empresas operadoras de missões, permitindo que todo o processo de busca, comparação, recomendação, reserva e acompanhamento da viagem seja realizado de forma integrada, segura e inteligente.

### 🎯 Objetivo Principal

A proposta da plataforma é funcionar como um **ecossistema completo de turismo espacial**, reunindo informações sobre destinos como:

- ✈️ **Voos Suborbitais** - Experiências de curta duração no espaço
- 🛰️ **Estações em Órbita Terrestre** - Hospedagem orbital
- 🌙 **Missões Lunares** - Viagens à Lua
- 🔴 **Programas de Hospedagem em Marte** - Experiências marcianas

### 💼 Funcionalidades Principais

- 🔍 **Busca e Comparação** - Encontre e compare experiências espaciais
- 🤖 **Inteligência Artificial** - Recomendações personalizadas de destinos
- 📅 **Gerenciamento de Reservas** - Sistema completo de booking e confirmação
- 👥 **Controle de Passageiros** - Gestão integrada de viajantes
- 📊 **Gestão de Capacidade** - Acompanhamento de disponibilidade das missões
- 💳 **Processamento de Pagamentos** - Sistema seguro de transações
- ⭐ **Avaliações Pós-Viagem** - Feedback dos usuários
- 📈 **Indicadores Operacionais** - Relatórios e análises de desempenho

## 🚀 Como Começar

### Pré-requisitos
- Node.js 14+
- npm ou yarn
- Oracle Database (para ambiente de produção)

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/gaaaabz/OrbitBookAPP.git
cd OrbitBookAPP
```

2. Instale as dependências:
```bash
npm install
```

3. Configure as variáveis de ambiente:
```bash
cp .env.example .env
```

4. Inicie o projeto:
```bash
npm start
```

## 📋 Escopo do Projeto

O OrbitBook abrange:

1. **Frontend** - Interface intuitiva e responsiva para reservas
2. **Backend** - APIs robustas para gerenciamento de dados
3. **Banco de Dados** - Oracle com automações em PL/SQL
4. **IA/ML** - Motor de recomendações personalizadas
5. **Segurança** - Validações e proteção de dados críticos
6. **Relatórios** - Indicadores operacionais e financeiros
